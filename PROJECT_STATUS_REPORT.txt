SUPPORT & TROUBLESHOOTING

Common Issues:

1. "adb not recognized"
   → Add SDK to PATH: setx ANDROID_SDK_ROOT "C:\...\Android\Sdk"

2. "JAVA_HOME error"
   → Clear it: set JAVA_HOME=

3. "Build fails"
   → Clean: .\gradlew.bat clean

4. "Device not found"
   → Check: adb devices

5. "Add button doesn't appear"
   → Reinstall: adb uninstall com.example.docexpiry

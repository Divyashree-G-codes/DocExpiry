GRADLE WRAPPER COMMANDS
=======================

Clean project:
  .\gradlew.bat clean

Build debug APK:
  .\gradlew.bat assembleDebug

Build release APK:
  .\gradlew.bat assembleRelease

Install debug APK:
  .\gradlew.bat installDebug

Install and run:
  .\gradlew.bat installDebug -wait
  adb shell am start -n com.example.docexpiry/.MainActivity

Run tests:
  .\gradlew.bat test

Check dependencies:
  .\gradlew.bat dependencies

Stop gradle daemon:
  .\gradlew.bat --stop

ADB COMMANDS
============

List connected devices:
  adb devices

Install APK:
  adb install -r app\build\outputs\apk\debug\app-debug.apk

Uninstall app:
  adb uninstall com.example.docexpiry

View logs:
  adb logcat
  adb logcat | findstr /I "docexpiry"
  adb logcat -c (clear logs)

Launch app:
  adb shell am start -n com.example.docexpiry/.MainActivity

Push file to device:
  adb push <local_file> /sdcard/<filename>

Pull file from device:
  adb pull /sdcard/<filename> <local_path>

Device shell:
  adb shell

TROUBLESHOOTING
===============

If 'adb' not found:
  setx ANDROID_SDK_ROOT "C:\Users\Shara\AppData\Local\Android\Sdk"
  Restart terminal

If JAVA_HOME error:
  set JAVA_HOME=
  .\gradlew.bat assembleDebug

If build fails:
  .\gradlew.bat clean
  .\gradlew.bat assembleDebug

If device not found:
  adb devices (check if device listed)
  Enable USB debugging on device
  Reinstall drivers if needed

If app crashes:
  adb logcat | findstr docexpiry
  Check build log for errors

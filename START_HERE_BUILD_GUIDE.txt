TROUBLESHOOTING (IF SOMETHING GOES WRONG)
================================================================================

‚ùå Error: "JAVA_HOME is set to an invalid directory"
‚úÖ FIX: Run these 2 commands:
   set JAVA_HOME=
   gradlew.bat clean assembleDebug

‚ùå Error: "gradlew.bat not recognized"
‚úÖ FIX: Make sure you're in the project directory:
   cd C:\Users\Shara\AndroidStudioProjects\DocExpiry

‚ùå Build stuck or slow
‚úÖ FIX: Stop gradle and try again:
   gradlew.bat --stop
   set JAVA_HOME=
   gradlew.bat clean assembleDebug

‚ùå APK not found after build
‚úÖ FIX: Check build log:
   type build_full.log | findstr /I "error"

‚ùå App doesn't install on phone
‚úÖ FIX: Enable USB debugging:
   Settings > About > tap Build Number 7 times
   Settings > Developer Options > USB Debugging ON

‚ùå Logo not showing on cards
‚úÖ FIX: Rebuild and reinstall:
   gradlew.bat clean assembleDebug
   adb uninstall com.example.docexpiry
   adb install -r app\build\outputs\apk\debug\app-debug.apk

‚ùå Add button doesn't work
‚úÖ FIX: Check if MainActivity.kt was updated:
   Search for "binding.fabAdd.setOnClickListener"
   If not found, rebuild: gradlew.bat clean assembleDebug

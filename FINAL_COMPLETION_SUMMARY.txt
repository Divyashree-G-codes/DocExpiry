TROUBLESHOOTING
================================================================================

Problem: Build fails with "JAVA_HOME is set to an invalid directory"
Solution:
  1. Run: set JAVA_HOME=
  2. Then: gradlew.bat clean assembleDebug

Problem: "gradlew.bat is not recognized"
Solution:
  1. Make sure you're in: C:\Users\Shara\AndroidStudioProjects\DocExpiry
  2. Try: .\gradlew.bat (with dot-slash)

Problem: APK installs but app crashes on launch
Solution:
  1. Connect device via USB
  2. Run: adb logcat | findstr docexpiry
  3. Look for error messages in output
  4. Check that Android 11+ is installed on device (API 30+)

Problem: Logo doesn't show on cards
Solution:
  1. Check that drawable files exist in: app/src/main/res/drawable/
  2. Verify names match in: CardTypeVisuals.kt
  3. Ensure XML drawable files are properly formatted

Problem: Add button doesn't work
Solution:
  1. Verify MainActivity.kt has FAB listener (lines 260-267)
  2. Check if AddEditCardActivity exists
  3. Try building again: gradlew.bat clean assembleDebug
